## üöÄ Getting Started

### Prerequisites

- **Node.js** 18+
- **npm** or **yarn**

### Installation

1. Clone the repository:
```bash
git clone https://github.com/chirag198/chirag-gulati-portfolio.git
cd chirag-gulati-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:4321/chirag-gulati-portfolio](http://localhost:4321/chirag-gulati-portfolio) in your browser

## üåê Deployment

The site is configured for GitHub Pages deployment:

```bash
npm run build
```

The built files will be in the `dist/` directory, ready for deployment.

### GitHub Pages Setup

1. Push your code to GitHub
2. Go to repository Settings > Pages
3. Set source to GitHub Actions or deploy the `dist` folder
4. Your site will be live at `https://chirag198.github.io/chirag-gulati-portfolio`
